# Алгоритми сортування - Лабораторна робота №1-8

Реалізація 8 алгоритмів сортування з інтерактивним меню та тестуванням продуктивності.

## Описання

Проект включає реалізацію наступних алгоритмів сортування:
1. **Selection Sort** - сортування вибором
2. **Insertion Sort** - сортування вставками
3. **Bubble Sort** - бульбашкове сортування
4. **Merge Sort** - сортування злиттям
5. **Quick Sort** - швидке сортування
6. **Shell Sort** - сортування Шелла
7. **Shaker Sort** - шейкерне сортування
8. **Heap Sort** - сортування з допомогою купи

## Функції

- 🎨 Кольорова візуалізація процесу сортування
- ⌨️ Інтерактивне меню (українська мова)
- 📊 Тестування продуктивності на масивах різного розміру
- ✏️ Введення масиву вручну або генерація випадкового масиву
- ⏱️ Виміривання часу виконання алгоритмів

## Користувацькі можливості

1. **Запустити окремий алгоритм** - виберіть цифру 1-8
2. **Тестувати всі алгоритми** - виберіть 9 і вибиріть рівень складності:
   - Легкий (10 елементів)
   - Складний (100 елементів)
   - Тяжкий (1000 елементів)
   - Дуже тяжкий (10000 елементів)
3. **Вихід** - виберіть 0
